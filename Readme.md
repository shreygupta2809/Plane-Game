# Top Gun Rescue

## Player Controls

| Key | Action        |
| --- | ------------- |
| W   | Go Forward    |
| A   | Go Left       |
| S   | Go Left       |
| D   | Go Rght       |
| L   | Shoot Missile |

---

## How to Run Game

```bash
$ npm i
$ npm start
```
